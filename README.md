# ADDJ Landing Page

This is a simple static landing page for ADDJ, a Boston-area DJ, karaoke, and event host brand.

## Files

- `index.html` holds the page content and section structure.
- `styles.css` controls the colors, layout, spacing, and mobile-friendly design.
- `script.js` adds the mobile menu behavior and updates the footer year automatically.

## How to Open the Site

1. Find the `index.html` file in this folder.
2. Double-click it to open it in your web browser.
3. You can refresh the browser after editing the files to see your changes.

No build tools, package installs, or frameworks are required.

## How to Edit the Site

Open the files in a code editor such as VS Code.

Common beginner edits:

- Change page text in `index.html`.
- Change colors near the top of `styles.css` inside the `:root` section.
- Replace `booking@example.com` in `index.html` with the real booking email.
- Add real images later by placing image files in the folder and using an `<img>` tag in `index.html`.

## Sections Included

- Hero section with ADDJ booking calls to action.
- Services section for DJ sets, karaoke, weddings, private parties, venue nights, and corporate events.
- "Curated for Your Crowd" section explaining custom music planning.
- "Pink Pony Night" featured section with the tagline "Boston's Newest Dance Party."
- Booking/contact section with email links.

## Notes

This version uses gradient visual sections as placeholders. Real photos, logo files, social links, and a working contact form can be added later.
